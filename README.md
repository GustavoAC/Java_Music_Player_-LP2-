# Java_Music_Player_-LP2-
Music Player implementado em Java com a biblioteca Javazoom JLayer

## Autores
- Gustavo Araujo Carvalho
- João Emmanuel Izidio da Silva

## Abrindo o projeto
O programa está pronto para ser aberto como um projeto comum da IDE Eclipse.
Vale ressaltar, porém, que a biblioteca jlayer, apesar de estar na mesma pasta
que o projeto, deve ser "linkada" manualmente pelo usuário.

## Execução
Quando executado pela primeira vez, o programa irá criar os arquivos necessários
para a execução e mostrar a tela de login.
O usuário admin está registrado por padrão com nome admin, id 0 e senha admin.

Na tela principal, é possível adicionar uma pasta ao programa pela opção na
parte inferior esquerda do programa. Após adicionada a pasta, um clique duplo
irá adicionar a música selecionada na playlist atual. De mesmo modo, basta um
clique duplo nas músicas da playlist atual para iniciar a execução da música.
Pressionando o botão inferior direito é possível salvar a playlist atual sob
um nome escolhido pelo usuário (somente VIPs). Dois cliques em alguma playlist
do usuário irá criar uma cópia da mesma para a playlist atual. Para modificar
alguma playlist do usuário, basta salvar a atual com o mesmo nome.

Vale lembrar que existem caixas de texto acima de cada uma das listas, essas
caixas servem como filtros que limitam as músicas somente àquelas que iniciam
com o seu conteúdo.

O botão de play tocará a música selecionada da playlist atual. Caso nenhuma
esteja selecionada, a primeira música será tocada. O botão de pause pausa e
despausa as músicas atuais, e o botão de stop para a execução completamente.
